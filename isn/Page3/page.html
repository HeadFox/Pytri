Mise en Place

Tout au long de l'année, nous avons peaufiné nos connaissances par le biais d'apprentissage de plusieurs languages, comme le Pyhton, l'HMTL, le CSS ou le PHP.
Nous devions donc faire un projet mêlant ces acquis que notre courageux professeur nous avait appris.
C'est pour cela que notre programme est entièrement fait en Python, et que notre site comporte les languages cités précédemment.
Les grandes raisons du choix de Python sont aussi: La stabilité du code, La facilité (Permettant d'éviter des erreurs fréquente) et aussi au fait que python n'a pas besoin de compilation et peut être utiliser sur quasiment tout les OS. Nous avons longtemps hésité sur le design du site, nous voulions quelque chose de présentable et visuellement plaisant à regarder.

Le design vient donc du site Codrops, qui est un site en libre accès, même pour les sites professionels. Nous avons ensuite modifié son code pour le modifier selon nos goûts.
Toutes nos Images ont été choisi avec soin et de préférence avec un fond transparent pour qu'elles soient mieux intégrer sur la page. Le site présente donc un accueil menant à d'autres pages.
Nous avons utiliser Notepad++ pour gérer le codage ainsi que APTANA STUDIO, un programme permettant de visualiser directement le code crée, ce qui a facilité nos corrections et la rapidité à modifier le design.
